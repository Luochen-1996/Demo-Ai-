提示词组装顺序
每次对话按以下顺序组装，Agent 工具定义在 system 后额外传入
1
全局底座提示词所有角色共用的 system，优先级最高
SYSTEM
2
角色设定附加在 system 末尾，定义角色性格与行为规则
CHAR
3
长期记忆摘要注入 system 末尾，描述用户偏好与历史事件
MEMORY
4
示例对话（Few-shot）system 之后的 user/assistant 消息对，引导语气
EXAMPLE
5
短期对话历史最近 N 轮真实对话
HISTORY
6
用户最新输入本轮 user 消息
USER
7
Agent 工具定义tools 字段传入，模型自主判断是否调用
TOOLS
